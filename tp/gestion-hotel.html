Contexte :

Ce Tp a constitué le quatrième chapitre du bloc de données de la première année de BTS SIO. Ce TP avait pour finalité la création de comptes utilisateurs et la gestion de leurs droits sur une base de données.

Description du TP :

Tout d'abord, les accès aux informations d’une base de données doivent être contrôlés pour des raisons de sécurité et de cohérence des données. Ainsi ce tp nous permet de nous exercer sur la création de compte utilisateurs avec leurs droits. Ce tp a été effectué sur papier puis en condition réelle sur WampServer pour les applications web.
