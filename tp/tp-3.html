Compétences associées

Contexte :

Ce tp a constitué le quatrième chapitre de programmation de la première année de BTS SIO. Ce TP avait pour finalité la manipulation de tableaux en C# ainsi que l'adaptation d'un code source pour répondre à une demande d’évolution.

Evolution d'un programme donnant des indicateurs sur des entrées :

Tout d'abord pour effectuer ce tp nous avons créé un nouveau projet vierge. Ensuite nous avons reçu diverses améliorations à effectuer avec sur ce code. Ainsi ce tp nous permet de manipuler les tableaux en C# pour les entrées et recettes. Pour finir, nous avons dû adapter pour chaque nouvelle amélioration le code source pour prendre en compte les demandes d'évolutions.
