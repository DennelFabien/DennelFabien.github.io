Concevez votre site web avec PHP et MySQL


PHP

(Hypertext Preprocessor)
PHP permet de développer toutes sortes de sites web : blogs, réseaux sociaux, sites e-commerce… car ce langage a justement été conçu pour créer des sites "vivants".

MySQL

(My Structured Query Language)
MySQL est un système de gestion de bases de données relationnelles. Il fait partie des logiciels de gestion de base de données les plus utilisés au monde3, autant par le grand public que par des professionnels, en concurrence avec Oracle, PostgreSQL et Microsoft SQL Server.

-Transmettre des données de page en page ;
